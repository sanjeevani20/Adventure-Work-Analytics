 Adventure Works Cycle Data Analytics Project

 Project Overview

This project focuses on analyzing Adventure Works Cycle sales and customer data to generate meaningful business insights. The analysis helps understand sales performance, customer behavior, product trends, and regional revenue distribution through interactive dashboards and KPI tracking.

 Tools & Technologies Used

* Excel
* Power BI
* Python
* SQL 

 Key KPIs

* Total Sales
* Total Profit
* Revenue Growth
* Customer Count
* Product Performance
* Regional Sales Analysis

 Features

* Interactive Dashboard
* Data Cleaning & Transformation
* KPI Cards
* Slicers and Filters
* Trend Analysis
* Business Insights Visualization

 Project Insights

* Identified top-performing products and regions
* Analyzed customer purchasing behavior
* Tracked monthly sales and profit trends
* Compared category-wise performance

 Files Included

* Dataset Files
* Dashboard Files
* Project Report
* Dashboard Screenshots

 Conclusion

This project demonstrates practical data analytics and dashboard development skills by converting raw business data into actionable insights for better decision-making.
